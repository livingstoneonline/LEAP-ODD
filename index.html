LEAP-ODD

Repository containing the LEAP ODD (and generated output) for the Livingstone-online Enrichment and Access Project

NOTE: All work or development happens on the https://github.com/LivingstoneOnline/LEAP-ODD/tree/gh-pages branch of this repository and thus is automatically live on the web at http://livingstoneonline.github.io/LEAP-ODD/index.html

File List:

  • index.html -- index file for site that just enables downloading of files
  • leap-doc.css -- CSS file for rendering the HTML documentation
  • leap.html -- HTML documentation generated from leap.odd.xml
  • leap.odd.xml -- LEAP TEI P5 ODD customisation and documentation file
  • leap.rng -- RNG schema generated from leap.odd.xml
  • leap-template-diaries.xml -- example template for LEAP diaries
  • leap-template-letters.xml -- example template for LEAP letters
  • leap.zip -- ZIP of the above files
  • Makefile -- Makefile used to generate the above files
  • profile -- TEI ODD transformation profile (just sets some basic parameters)
  • README.md -- This readme file
